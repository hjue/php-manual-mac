## EverDoc

Everdoc - 离线文档浏览器

- 目前支持的开发文档: Nodejs,PHP,MySQL,Redis,Javascript, jQuery, jQuery UI, CSS,更多开发文档正在制作中
- 可以按方法名、类名等快速搜索
- 支持指定文档内搜索，如在PHP文档内搜索，可输入php:str_rep
